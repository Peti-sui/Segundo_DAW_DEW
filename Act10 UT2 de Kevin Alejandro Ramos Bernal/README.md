

**Funcionalidad:**  
La función `multiplicarArray()` recibe un array de números enteros, identifica el número menor y el mayor, y retorna el resultado de multiplicarlos. El usuario ingresa los números en un campo de texto separados por comas y, al hacer clic en un botón, se muestra el resultado en la página de forma dinámica.

**Ejemplo:**  
Entrada: `3, 9, 1, 5`  
Salida: `El número menor es 1, el mayor es 9, y su multiplicación es 9.`

---

**Functionality:**  
The function `multiplicarArray()` receives an array of integers, finds the smallest and largest numbers, and returns their multiplication. The user enters the numbers in a text field separated by commas, and by clicking a button, the result is dynamically displayed on the page.

**Example:**  
Input: `3, 9, 1, 5`  
Output: `The smallest number is 1, the largest is 9, and their multiplication is 9.`
